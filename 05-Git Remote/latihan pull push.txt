bagimana pull push bekerja

- membuat push tidak bekerja
1. buat file delete
2. commit tetapi jangan di push
3. buka repository di github
4. hapus file delete
5. buka terminal git push

- menyelesaikan problem delete data di github
1. buka git graph
2. klik branch lokal yang berwarna merah (git)
   * file di github akan menjadi branch utama
3. klik kanan checkout branch, tulis nama menjadi eror
4. ketik git checkout eror
5. ketik merge
6. ketik git branch -d master
7. ketik git branch master
8. ketik git checkout master
9. ketik git branch -d eror
